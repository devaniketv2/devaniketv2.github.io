Core Technology

The cutting-edge stack behind ForgeAI Studio — self-hosted, private, and production-grade.

AI & Models

  • Ollama — local LLM runtime; your models run on your hardware
  • Multi-provider cloud models — optional escape hatch per chat, with routing and defaults
  • Whisper — local speech-to-text in 50+ languages
  • Kokoro + Piper — natural text-to-speech, language-aware engine routing
  • Embedding & reranking models — semantic retrieval with cross-encoder precision

Application

  • Spring Boot 3 · Java 21 — modular backend with domain-driven modules
  • Angular 19 — standalone components, signals, full RTL support
  • Versioned service API — scoped API keys let other systems consume the engine

Data

  • PostgreSQL 16 + pgvector — records and vector search in one durable store
  • MinIO — S3-compatible object storage for documents
  • Flyway — versioned, auditable schema migrations

Operations & Security

  • Docker Compose — the entire stack as declarative services
  • Prometheus + Grafana — metrics, dashboards and alerts
  • JWT + RBAC + audit — fine-grained authorities, append-only audit trail
